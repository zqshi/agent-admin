**文档目标：** 项目入口文档，提供完整的平台概述和导航

## 📋 项目简介

KingSoft是一个企业级的数字员工智能管理平台，基于LLM+MCP架构，为数字员工提供实时监控、数据分析、A/B测试和全生命周期管理。

### 🎯 核心价值
- **可视化监控**：实时追踪数字员工服务状态
- **科学决策**：基于贝叶斯统计的A/B测试
- **智能优化**：可解释性AI驱动的持续改进
- **成本控制**：精细化的Token使用和成本分析
